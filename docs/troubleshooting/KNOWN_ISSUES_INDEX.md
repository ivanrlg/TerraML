## Como Documentar un Issue

Cuando encuentres un issue recurrente, crea un archivo en esta carpeta:

```markdown

Descripcion del problema y como se manifiesta.

## Causa Raiz
Por que ocurre.

## Solucion
Pasos para resolver.

## Archivos Afectados
- `ruta/al/archivo.cs`

## Prevencion
Como evitar que vuelva a ocurrir.
```
